# Eldritch User stories

## CORE

### Home screen

#### Must have

As a user, I want to be able to play as a guest

As a user, I want to be able to choose a character
As a user, I want to be able to start a solo game
As a user, I want to be able to create a room
As a user, I want to be able to join a room

#### To implement later on

As a user, I want the option to log in
As a user, I want to choose a difficulty level
As a user, I want to choose a question pack

### Encounter page

#### Must have

As a user, I want to see which answer I have voted on
As a user, I want to see the damage I have done/taken
As a user, I want to see my/the monsters total health
As a user, I want to see the correct answer when voting has finished
As a user, I want to see what other players have voted on
As a user, I want to know when I have won/failed
As a user, I expect a fair difficulty level
As a user, I expect answers to be correct

#### To implement later on

As a user, I want to be able to change my answer

## STRETCH GOALS

### Stretch goal priority list

1. Timer
2. Character sheet
3. Save progress
4. Loot box
5. Mobile
6. Leaderboard
7. Music/sfx
8. Storyline
9. Add quiz

### General

As a user, I want to be able to save my progress
As a user, I want to play the game on my mobile device or browser
As a user, I want an engaging storyline
As a user, I want appropriate music/sfx

### Encounter page

As a user, I want to see how much time I have to answer a question
As a user, I want reactive difficulty based on my performance
As a user, I want to see the difference in a boss monster

### Loot box

As a user, I want to vote on a loot box
As a user, I want to see the result of my choice
As a user, I want the item to be stored in my inventory
As a user, I want voting to finish in a timely manner

### Character Sheet

As a user, I want to see my character sheet
As a user, I want to see what items I have equipped
As a user, I want to change my equipment from my inventory
As a user, I want to see how many quizzes I have attempted
As a user, I want to see how many quizzes I’ve passed/failed
As a user, I want to see how many questions I’ve answered
As a user, I want to see how accurate I have been with my answers
As a user, I want to gain experience and levels
As a user, I want my stats to increase with my level
As a user, I want to see my stats (atk, def, san)
As a user, I want to understand how each stat affects the game
As a user, I want to see how much xp I need to level up

### Leaderboard

As a user, I want to see the top 10 on a leaderboard
As a user, I want to see my position on a leaderboard
As a user, I want to be rewarded for doing well
As a user, I want to see a weekly/all time leaderboard

### Add quiz

As a user, I want to be able to upload a quiz
As a user, I want to be rewarded for uploading a quiz
